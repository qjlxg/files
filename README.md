# 📈 2025 极致缩量低吸系统

本系统用于量化筛选“极度超跌 + 卖盘枯竭”的标的，通过捕捉筹码真空期的反转机会实现获利。

---

### 🚦 实战操作信号指南
> **重要：** 筛选出标的后，请按以下信号执行操作，严禁盲目直接买入。

| 信号类型 | 关键特征描述 | 实战动作建议 |
| :--- | :--- | :--- |
| **🟢 最佳介入点** | 股价收阳，收盘价**向上突破**前一交易日最高点 | **果断建仓 (首仓)** |
| **🔥 量能确认** | 分时放量，今日量数从 0.8 以下**跃升至 1.2 以上** | **择机加仓** |
| **🔴 失败信号** | 股价放量**跌破 5 日均线 (MA5)** 且收盘未收回 | **立即止损 / 撤离** |

---

### ⚙️ 核心参数调优说明
| 参数名称 | 默认值 | 调整建议 | 预期效果 |
| :--- | :--- | :--- | :--- |
| **MIN_PRICE** | 5.0 | 调高至 10.0 | 锁定中大盘绩优股，过滤低价股 |
| **MAX_VOLUME_RATIO** | 1.2 | 调低至 0.8 | 筛选出的筹码更加“静默” |
| **RSI6_MAX** | 30 | 调低至 25 | 选股更严苛，仅捕捉深度超跌 |
| **MAX_TODAY_CHANGE** | 1.5 | 调高至 2.5 | 普涨行情下避免漏掉领头羊 |

---

### 💡 复盘核心逻辑
1. **先看战绩**：优先关注“连跌5天”以上且今天收红的票，弹性最大。
2. **后看状态**：显示 **"⭐突破/放量"** 的标的具备极高的短线起飞概率。
3. **空间原则**：距 60 日线空间必须 > 10%，否则盈亏比不划算。
